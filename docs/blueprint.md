# **App Name**: CityZen Reporting

## Core Features:

- User Authentication: Secure user authentication using Aadhaar and email verification to ensure data integrity and prevent spam.
- Problem Reporting: Allow users to submit photos of civic issues (e.g., potholes, broken streetlights) with live location data.
- Issue Prioritization Tool: AI tool that prioritizes reported issues based on severity, frequency, and location to guide municipal cooperation efforts.
- Municipal Integration: Interface for municipal cooperation to receive, review, and manage reported issues.
- Resolution Tracking: Enable the municipal corporation to update the resolution process and provide estimated resolution times. Also allow users to track the process.
- Admin Dashboard: A dashboard to visualize reported problems, resolution status, and key performance indicators.

## Style Guidelines:

- Primary color: A vibrant blue (#29ABE2) to represent trust and civic responsibility.
- Background color: Light gray (#F5F5F5) for a clean, accessible user interface.
- Accent color: An energetic orange (#FF9933) for calls to action and progress indicators.
- Body and headline font: 'PT Sans' for readability and a modern, approachable aesthetic.
- Use clear, representative icons for different issue types (e.g., potholes, streetlights).
- Clean and intuitive layout optimized for mobile devices with easy access to reporting features.
- Subtle animations for progress updates and confirmations to provide user feedback.
